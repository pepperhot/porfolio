BOT TELEGRAM: GENERATEUR TIKTOK

Objectif : Automatisation complète de création de contenu viral (Karaoké & Lyrics) depuis Telegram via IA.F

💡 Fonctionnalités Clés :

  • Mode Lyrics : Scraping de paroles & pochettes, génération d'images stylisées (Glitch/Particules) via Pillow.
  • Mode Karaoké : Téléchargement YouTube, extraction audio, transcription IA via OpenAI Whisper pour le timestamping, rendu vidéo via MoviePy.
  • UX/UI : Interface Telegram interactive, personnalisation des polices et thèmes.

🛠️ Défis Techniques & Compétences :

Intelligence Artificielle

Intégration de faster_whisper (ASR) pour une transcription audio ultra-précise et synchronisée.

Traitement Média (Image/Vidéo)

Pipeline de rendu 100% Python. Calculs de coordonnées, filtres pixel par pixel et compositing sans logiciel externe (MoviePy, Pillow).

Architecture & Async

Architecture modulaire et utilisation de asyncio pour gérer les tâches lourdes (rendu vidéo) sans bloquer le bot.

GUIDE D'UTILISATION

01. DÉMARRAGE

Tout commence par la commande /start. Le bot s'initialise et vous présente le Menu Principal.

Depuis ce menu, vous pouvez :

  • Choisir un parametrage couleur ou type de resultat
  • Sélectionner le mode de création (Lyrics ou Karaoké).

02. MODE LYRICS

Idéal pour obtenir rapidement les paroles d'une chanson avec un style visuel.

Action : Cliquez sur le bouton "Lyrics" puis envoyez simplement le Titre de la chanson (ex: "Eminem:Godzilla").

Résultat : Le bot renvoie les paroles complètes et génère deux image "Cover" unique et une description (près pour etre posté).

03. MODE KARAOKÉ

Le mode le plus avancé. Il transforme une vidéo YouTube en clip vidéo avec paroles synchronisées.

Action : Cliquez sur "Karaoké" puis envoyez un Lien YouTube ou le meme shema pour le lyrics Eminem:Godzilla.

Résultat : Le bot télécharge, analyse l'audio, synchronise le texte et vous envoie le fichier vidéo final.

YHACK: PLATEFORME D'AUDIT DE SÉCURITÉ

Description : Yhack est un scanner de vulnérabilités web automatisé (DAST - Dynamic Application Security Testing) capable d'auditer la sécurité des sites en temps réel, de la reconnaissance passive à l'exploitation active.

🛡️ Modules d'Attaque Implémentés :

  • Injection SQL (SQLi) : Fuzzing paramétrique et détection d'erreurs SGBD spécifiques.
  • XSS & RCE : Payloads polyglottes pour tester la réflexion et l'exécution de commandes système.
  • SSTI (Mako) : Injection de templates hybride via Puppeteer pour l'automatisation du navigateur.
  • Brute-Force : Attaque dictionnaire intelligente avec détection de Rate Limiting.
  • Reconnaissance : Scan de ports, énumération de sous-domaines (crt.sh) et détection de CMS.

🛠️ Architecture Technique :

Moteur Backend (Node.js)

API REST asynchrone utilisant node-fetch pour les requêtes HTTP brutes et gestion modulaire des scans.

Browser Automation

Utilisation de Puppeteer (Headless Chrome) pour les interactions complexes (remplissage formulaires, exécution JS).

Interface & Rapport

Frontend "Dark Mode" avec feedback console en temps réel et génération de rapports de scan dynamiques.

WORKFLOW D'AUDIT

01. CIBLAGE

Configuration de l'audit via le tableau de bord.

Action : Entrez l'URL cible (ex: http://vulnerable-app.com) et sélectionnez les modules d'attaque à activer (SQLi, XSS, Full Scan).

Le système vérifie la disponibilité de la cible avant de lancer le moteur.

02. EXÉCUTION DU SCAN

Le moteur Yhack entre en action.

Back-end : Le serveur lance les scripts Puppeteer pour simuler un utilisateur réel et injecte les payloads via l'API.

Visuel : La console en temps réel affiche les tentatives d'intrusion et les réponses HTTP (200, 403, 500).

03. RAPPORT & FLAGS

Analyse finale des résultats.

Une fois le scan terminé, Yhack génère un rapport classant les vulnérabilités par criticité (Low, Medium, Critical).

En mode CTF, le scanner extrait automatiquement les Flags trouvés dans la base de données ou les fichiers systèmes.

PROJECT MATRIX : RPG GO 2D

Description : Un jeu de combat et de gestion en 2D développé en Go avec la bibliothèque Ebiten. Plongez dans l'univers de Matrix, gérez votre économie, améliorez vos compétences et affrontez des ennemis dans des lieux emblématiques.

🎮 Gameplay & Fonctionnalités :

  • Exploration : Navigation via une carte interactive (Dojo, Mall, Place, Building) débloquée progressivement.
  • Le Dojo & La Forge : Zone d'entraînement et boutique d'améliorations. Achetez des techniques (Lutte, Karaté) et équipements.
  • Système de Combat : Affrontements au tour par tour basés sur une variante complexe du "Pierre-Papier-Ciseaux" (Boxe, Judo, Jujutsu, Karate, Lutte).
  • Consommables : Achat stratégique de "Pilules Rouges" (Bonus Victoire) ou "Pilules Bleues" (Handicap Ennemi) avant les combats.

🛠️ Architecture Technique :

Le projet est structuré autour d'une boucle de jeu "Update/Draw" performante.

Machine à États (State Machine)

Gestion dynamique des scènes (currentState) permettant des transitions fluides entre Menu, Combat, Boutique et Exploration.

Moteur Ebiten (Go)

Utilisation de la librairie 2D Ebiten pour le rendu graphique, la gestion des inputs (Souris/Clavier) et l'audio.

Logique Combat & IA

Algorithme de résolution de conflits (5 arts martiaux) et bot adverse avec choix aléatoires pondérés.

💾 Gestion de Données :

  • Économie Persistante : Système monétaire in-game pour l'achat d'upgrades.
  • Backdoor Développeur : "God Mode" caché activable par interaction spécifique (Zone secrète) pour débloquer tout le contenu pour les tests.

MON HOMELAB

Objectif : Créer un environnement réseau souverain, sécurisé et performant pour l'auto-hébergement de services et l'apprentissage réseaux/systèmes.

🔌 Hardware & Réseau :

  • Switch Administrable : Gestion du trafic et segmentation du réseau (VLANs IoT, Invité, Serveurs).
  • NAS (Network Attached Storage) : Stockage centralisé des données, configuration RAID pour la tolérance aux pannes et serveur de sauvegardes.

Réseau Physique

Dépannage reseaux 2.4Ghz, 5Ghz. Attribution des ports du routeur. Bloquer les ports à risque.

Distribution

Donner des points d'accès réseau en VLAN ou LAN. CABLE MANAGEMENT.

Le Raspberry Pi agit comme le gardien basse consommation du réseau, exécutant des services critiques 24/7.

🛡️ Sécurité & Réseau (Raspberry Pi) :

  • AdBlock Réseau Pi-hole : Serveur DNS local qui filtre les requêtes publicitaires et les traceurs pour tous les appareils de la maison (PC, Téléphone, TV).
  • VPN (WireGuard / OpenVPN) : Tunnel sécurisé permettant d'accéder au réseau local et aux services depuis n'importe où dans le monde de manière chiffrée.
  • Reverse Proxy : Gestion des accès HTTPS et des sous-domaines locaux.

🚀 Autres Services Hébergés :

Stokage

Augmentation du stokage pour les petits appareils (telephone | dossier sensible) et securisation des données.

Site web

Sites pour competions de babyfoot avec une base de données dédiée.

EXTRACTEUR D'AIR & FILTRATION

Objectif : Créer un appareilles portables de filtration d'air fumées de soudure, plastique entièrement customisables et imprimables en 3D.

🛠️ Design & Fabrication :

  • Modélisation Paramétrique : Développement de boîtiers en PLA pour faire circuler le flux d'air et la facilité d'assemblage.
  • Assemblage : Conception en plusieurs parties 2 faces (avant , arriere) soudure, collage, aimant de fixation.
  • Filtration : Intégration de filtres à charbon actif interchangeables pour neutraliser les odeurs et particules fines.

⚡ Prototype Avancé (V2) :

Le modèle le plus complexe, ventilateur plus puissant et plus d'autonomie.

Gestion Énergie (12V)

Alimentation via Batterie 12V avec module de charge USB-C intégré pour une recharge universelle.

Contrôle & Monitoring

Intégration d'un multimètre numérique pour l'état de la batterie et interrupteur On/Off industriel.

🔧 Composants Clés :

  • Ventilateur de PC axial 12V.
  • BMS (Battery Management System) pour la sécurité charge/décharge.
  • Câblage et soudure interne .
